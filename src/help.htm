LeMMA - a GUI frontend for creating MMA files

Created by Gek S. Low. Released under the GPL.

Note that this is NOT really an MMA editor, but rather a simple front-end to generate simple MMA files. I wrote this program so that I can churn out and play chord progressions easily.

BEFORE YOU START

  1. If you just installed a new version of LeMMA and it exits immediately, you may need to delete settings.dat and grooves.dat because of changes in the file formats.
  2. Please click on "Settings" to set up the right paths. You may want to refresh the grooves library in "Settings" which will read the groove information in your MMA installation.
  3. Custom groove library is optional and is for advanced users who have set up their own grooves in another location. It uses the SetLibPath and SetAutoLibPath directives. Please refer to MMA documentation. Note that this feature is still fairly experimental.

USAGE

  1. You can supply a filename to LeMMA and it will try to open the file.
  2. If you are having problems, start LeMMA at the console with the '--debug' option.
  3. Just type your chords in the measure boxes (e.g. 'C / F /'. You are currently limited to up to 80 measures.
  4. For repeats, click on the barlines to toggle to the various types.
  5. Type '[1', '[2', etc in the measure boxes for repeat endings 1, 2, etc (the numbers don't matter actually)
  6. Please note that the first empty measure is regarded as the end of the song. All following measures will be ignored!
  7. To have a measure continue the last chord in the previous measure, enter '/'
  8. LeMMA doesn't interpret the chords/contents in the measures in any way. Enter garbage and you'll get garbage. :-)
  9. You can set the groove for a measure by clicking on '(Groove)' above the measure.
  10. Advanced users can click on 'Code' to insert MMA commands for each measure. The commands are inserted between the Groove command and the measure definition. Use 'View MMA' to preview the generated codes.
  11. You can load and save MMA files, but note that this is meant for files generated by LeMMA. If you save/load a carefully handcrafted MMA file you probably WILL lose information in the process.
